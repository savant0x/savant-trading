## New lessons (this period, unverified at the 3-cycle mark)

### LESSON-009 (candidate, 1 cycle): Decode the on-chain revert BEFORE the docs

When debugging a swap revert, run the 4-byte selector lookup first. The chain is the
source of truth. Docs describe intent; the revert describes reality. FID-157/158
demonstrated the failure mode: I read the 0x docs and assumed allowance was the issue
based on `issues.allowance` in the /quote response. The actual revert was
`ECDSAInvalidSignatureLength(64)` — signature format, not allowance.

**Evidence:** FID-158 line 22-28 documents my own wrong diagnosis. The selector
`0xfce698f7` was in front of me from FID-157 line 14. I didn't run the lookup.

**Promotion rule:** 1 cycle observed. Needs 2 more (FID-160, FID-163, FID-164 — three
independent verifications) to graduate to MEMORY.md. Will track.

### LESSON-010 (candidate, 1 cycle): Out-of-scope requires a specific reason that survives strict-read

FID-163 Loop 5/6 expanded scope multiple times because "out of scope" claims kept
collapsing under strict-reading. The only valid reasons for LLM-bound data to be
excluded are: (a) it doesn't reach the LLM, or (b) it's a display surface for humans.
"Future use," "might wire it later," "dead code anyway" — none of these are valid.

**Evidence:** FID-163 line 100-114 ("Spencer's Rule"), Loop 5/6 self-corrections
(line 322-332). The 4 bug classes were all "out of scope" until I strict-read the
rule.

**Promotion rule:** 1 cycle observed. Needs 2 more to graduate.

### LESSON-011 (candidate, 1 cycle): Use temp .ps1 files to avoid cmd caret-escaping

start.bat's pre-build cleanup required 3 PowerShell commands with chained pipes,
filters, and redirections. PowerShell inside cmd requires `^|` and `^>` escape
sequences that are unreadable and error-prone. The correct pattern: write the PS
command to a temp .ps1 file (`%TEMP%\savant_prebuild_%RANDOM%.ps1`), execute via
`powershell -NoProfile -ExecutionPolicy Bypass -File`, read output. Avoids all
escape hell.

**Evidence:** start.bat lines 41-49 (savant block). Worked first try, no debugging.

**Promotion rule:** 1 cycle observed. Needs 2 more to graduate.

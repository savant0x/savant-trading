## Lessons (in LEARNINGS.md)

15 new lessons captured. Top 3:

1. **Per-pair state isolation is the right pattern for batch LLM loops.** Singleton state across distinct items = cross-contamination bug. Pattern: `HashMap<item_key, ItemState>`.
2. **Decode the actual error before forming a hypothesis.** 504 wasn't in the retry list. 1-character fix.
3. **Reuse existing utilities before writing new code.** `token_budget::count_tokens` already wraps tiktoken. Saved ~25 lines.

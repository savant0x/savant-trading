## Lessons Learned

- Always add timeouts to network calls — `reqwest::Client::new()` has no default timeout
- Gas prices are stale by the time a tx is broadcast — always add a 50% buffer
- `tracing` deadlocks with `RwLock` when both write to the same subscriber — use `eprintln!` for critical paths
- Single source of truth for logging prevents format drift across the codebase
- The AI is correctly disciplined — it waits for valid setups instead of forcing trades

## Lessons Learned

- `reqwest` timeout doesn't cover DNS/TLS hangs — always add `tokio::time::timeout` around external API calls
- The 0x API on Arbitrum is intermittently unreliable — need fallback to 1inch
- Full merge of divergent branches is risky — cherry-picking is safer
- ECHO Protocol /dev folder must be maintained every session — FIDs archived, LEARNINGS updated, session summary created
- ANSI color codes from tracing subscriber can bleed into custom log output — disable tracing colors when using custom logging
- CoinGecko free API gives 5m candles for 1 day (289 price points) — sufficient for AI analysis
- Arbitrum has 700K+ tokens but only ~100 with >$1M volume — use Blockscot API for discovery
- Runtime token DB extension needed for discovered addresses — static OnceLock is immutable
- Process panics in HTTP client can kill the entire engine — need catch_unwind around external API calls
